# vue-element-admin

一个简易的Vue3 + Vite + TypeScript + Element Plus 后台管理系统模板

## 启动项

```sh
pnpm install
```

### 开发模式

```sh
pnpm dev
```

### 打包

```sh
pnpm build
```

### 运行单元测试

```sh
pnpm test:unit
```

### 运行端到端测试

```sh
pnpm test:e2e:dev
```

发布生产前进行端到端测试

```sh
pnpm build
pnpm test:e2e
```

### 代码格式检验

```sh
pnpm lint
```
